Photography Tools

These are all the tools I have used in photography in my classwork and own personal projects.

My Cameras

Sony α6400

This is my main camera, more for when I plan to take photos.
I use the base kit 18mm - 135mm lens as well as a Tamron 70mm - 300mm lens for my various shots.

iPhones

When I do not have easy access to my Sony, I usually rely on my phone camera to try to get a shot.
When I really started photography, I had an iPhone XS Max. Now, I have an iPhone 13 Pro Max.

Somehow, some of my best shots have come from these phones.

Software

Lightroom Classic

This is my main editing software. I have become comfortable with the workflow in this software, so it is the easiest for me.

Photoshop

I have not used Photoshop as much, but I can use this software for any tweaks that cannot be done in Lightroom.
